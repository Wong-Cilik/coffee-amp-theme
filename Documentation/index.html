Descripton:

Coffee AMP is a Accelerated Mobile Pages (AMP) for Blogger Template. It's a simple design, just few seconds to load and get everything running. It's just share for free. This theme support with microforats, microformats 2, microdata and responsive design.

Installation

When you are ready to install a theme, you must first upload the theme file. The theme file can be uploaded in one way .

a. Upload File

  1. Open the file.
  2. Open the XML file with a text editor (Notepad or something like that). Then select all text and copy (Ctrl+A, Ctrl+C in Window).
  3. Return to your blog dashboard, access your Template menu, click Edit HTML
  4. In Editor HTML window, select all old code and paste the new code that you copied from the release file into code field. Then, click Save template.

b. Blog Setting (Activate Blog Post Type)

This step to make blog post type enable.

  1. Access your blog dashboard > Settings > Other. In Site feed - Allow Blog Feed choose Full.

  2. then Enable Title Links and Enclosure Links change to Yes

  3. Then Click Save settings.

d. Setting Meta Tag and Open Graph

  1. Find on Head tag and replace XXXX with your google plus link and your name on meta me author. 
    <meta content='XXXXX' name='me Author'/>
<link href='https://plus.google.com/XXXX' rel='author me'/>
<link href='https://plus.google.com/XXXX' rel='publisher'/>
  2. Find on Head tag and replace XXXX with your google site verification. 
    <meta content='XXXX' name='google-site-verification'/>
  3. Find on Head tag and <b:includable id='post' var='post'> then replace XXXX with your fb profile id, locate on find my id and if you have fb:app_id just change the fb:app_id value or you can create one at Developer Facebook. 
    <meta content='XXXX' property='fb:admins'/>
<meta content='XXXX' property='fb:profile_id'/>
<meta content='1429806127066799' property='fb:app_id'/>
  4. Find on Head tag and <b:includable id='post' var='post'> then replace XXXX with your twitter account. 
    <meta content='@XXXXX' name='twitter:creator'/>

    Note: Before your cards show on Twitter, you must first have your domain approved. Fortunately, it's a super-easy process. After you implement your cards, simply enter your sample URL into the validation tool. After checking your markup, select the "Submit for Approval" button. Go Here : Cards Validator for testing and approval

  5. Find on includable <b:includable id='shareButtons' var='post'> then replace with your fb:app_id or you can leave it default. 
    <amp-social-share data-param-app_id='1429806127066799' height='35' type='facebook' width='35'/>

Customizations

Every Make New Post

  1. Delete this code, every make new post: 
    <div dir="ltr" style="text-align: left;" trbidi="on">
  2. embed <noscript> </noscript> this code, every first image post or page example: 
    <noscript><img height='282' src='https://1.bp.blogspot.com/-mo2fFID_mYk/WaSg5Aq0DeI/AAAAAAAACxw/mTJ6XNdoAzgPTAFzF-WMzMgcJ-EC5TYZwCEwYBhgL/s640/AMP_WordPress_AMP_Project1.jpg' width='640'/></noscript>
  3. embed <amp-img> </amp-img> this code, when insert image post or page example: 
    <amp-img height='282' layout='responsive' src='https://1.bp.blogspot.com/-mo2fFID_mYk/WaSg5Aq0DeI/AAAAAAAACxw/mTJ6XNdoAzgPTAFzF-WMzMgcJ-EC5TYZwCEwYBhgL/s640/AMP_WordPress_AMP_Project1.jpg' width='640'></amp-img>
  4. embed <amp-youtube> </amp-youtube> this code, when embed video from youtube at post or page example: 
    <amp-youtube data-videoid='5Yjoe54vzwE' height='270' layout='responsive' width='480'></amp-youtube>
